# CodeAlpha_DiseasePrediction

### Disease Prediction Using Machine Learning

###  Project Overview
This project predicts whether a breast cancer tumor is benign or malignant using a Machine Learning model. The model is trained on the Scikit-learn Breast Cancer Dataset and achieves high prediction accuracy.

###  Objective
To develop a disease prediction system that can classify breast cancer cases accurately using machine learning techniques.

### Dataset
- Scikit-learn Breast Cancer Dataset
- Total Samples: 569
- Features: 30
- Classes: Benign and Malignant

### Algorithm Used
- Random Forest Classifier

###  Technologies Used
- Python
- Scikit-learn
- NumPy

### Model Accuracy
**96.49%**
